## API Cascade Path

`include/amd_smi/amdsmi.h` → `src/amd_smi/amd_smi.cc` → `tools/generator.py` → `py-interface/amdsmi_wrapper.py` → `py-interface/amdsmi_interface.py` → `amdsmi_cli/amdsmi_commands.py` → `docs/`
