# AMD SMI Python Tests

Python `unittest`-based suite. For the full layout, the three runners
(`unit_tests.py`, `integration_test.py`, `cli_unit_test.py`), how to run them,
filtering options, and the installed paths, see the test design guide:

[../../docs/conceptual/test-design.md](../../docs/conceptual/test-design.md)

## Prerequisites

- **Python 3.8+**
- The **`amdsmi` Python package installed** — the integration and CLI tests import it.
  See the [AMD SMI installation guide](https://rocm.docs.amd.com/projects/amdsmi/en/latest/).

## Reference

- Python `unittest` framework: <https://docs.python.org/3.8/library/unittest.html>
